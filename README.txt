         ----- The Unboost C++ Library -----
      ----- Time to say good-bye to Boost -----

This is the Unboost C++ Library, that is a convenience compatibility 
library for C++.

The Boost C++ Libraries (http://www.boost.org) are famous and very 
sophisticated but too large and expensive for daily use.

This Unboost (just 104KB) enables you to avoid that expensive booster 
if possible on your C++ compiler (C++11, std::tr1, substitute tiny
implementations).

See source for more details.
